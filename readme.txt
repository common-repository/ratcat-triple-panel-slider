== Installation ==

= Minimum Requirements =

* WordPress 3.5 or greater

= Automatic installation =

Automatic installation is the easiest option as WordPress handles the file transfers itself and you don’t even need to leave your web browser. To do an automatic install of RATCAT Triple Panel Slider, log in to your WordPress admin panel, navigate to the Plugins menu and click Add New.

In the search field type “RATCAT Triple Panel Slider” and click Search Plugins. Once you’ve found the plugin you can install it by simply clicking Install Now. After clicking that link you will be asked if you’re sure you want to install the plugin. Click yes and WordPress will automatically complete the installation.

= Manual installation =

The manual installation method involves downloading “RATCAT Triple Panel Slider” plugin and uploading it to your webserver via FTP application.

1. Download the plugin file to your computer and unzip it
2. Using an FTP program, or your hosting control panel, upload the unzipped plugin folder to your WordPress installation’s wp-content/plugins/ directory.
3. Activate the plugin from the Plugins menu within the WordPress admin.
